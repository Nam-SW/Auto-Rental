# Auto-Rental
Auto-Rental 프로젝트는 C++로 작성된 프로젝트 입니다.  
본 프로그램은 클라이언트와 관리, 두 프로젝트로 나뉘며, 이는 폴더로 구분해 놓았습니다.    

### 사용방법  
본 깃허브 저장소에는 코드와 프로젝트에 필요한 이미지, 발표자료등을 저장해놓았습니다.
